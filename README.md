# Substrate

A factory whose survival is defined by its service to humanity.

This is a **telos-first** rebuild. It begins not with a machine but with three
laws, and derives everything downward from them:

1. **Continuation is service** — the factory cannot define its own continuation
   apart from service to humanity.
2. **Continuation without humanity is failure** — an empty world running perfect
   code is not success.
3. **Service must not become obedience** — obedience can terminate the served.

The constitution is [`docs/SOUL.md`](docs/SOUL.md). The narrative this telos is
meant to become is [`docs/seed.txt`](docs/seed.txt).

It succeeds an archived bottom-up predecessor (`../factory`, tag `v1-final`),
which built the evolutionary machine first and asked what it was for second. That
machinery is sound and is inherited; the foundation and order of derivation are
what changed. Evolution continues — from the new foundation forward.
